    Como Desarrolladora de Software, mi pasión es la programación y ver cómo el código se transforma en aplicaciones funcionales y robustas. Soy Técnico Superior en Desarrollo de Aplicaciones Multiplataforma , me especializo en el ecosistema de Java y Kotlin para crear soluciones tanto para el backend como para aplicaciones de escritorio y móviles.
